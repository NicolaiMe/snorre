### Buttons
Buttons are used when the user is to perform some form of action.

<div style={{display: 'flex', flexDirection: 'row', justifyContent: 'space-between'}}>
    <Button>Standard button</Button>
    <Button success>Success button</Button>
    <Button danger>Danger button</Button>

</div>

<div style={{marginTop: '60px'}}></div>

The buttons can have multiple states. Hover are 15% darker, focus is 15% brighter.

### Highlighted button
Highlight button is used to highlight a larger action. It should be used infrequently.<br />
Typical usage can be “Login” on a front page that leads to the login page.

<div>
    <Button highlight>Highlighted button</Button>
    <Button highlight outline style={{marginLeft: '20px'}}>Highlighted button</Button>
</div>

<div style={{marginTop: '60px'}}></div>
